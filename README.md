# OSDT Weekly

## 简介

开源开发工具周报（OSDT Weekly）是由HelloGCC及HelloLLVM社区发起和维护的摘要类周刊。内容覆盖GCC、LLVM、GDB、Binutils、TVM、QEMU等开发人员常用的开源项目。每周三晚上发布。内容通过Git仓库托管在HelloGCC的GitHub账号里，欢迎提交PR :-)

仓库地址： https://github.com/hellogcc/osdt-weekly/
